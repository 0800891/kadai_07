## ⑤工夫した点・こだわった点
- select.phpにおいてdelete機能入れた。deleteするとidが再度１から振り直される。
- 新しくデータを追加すると、idが、変な数字で始まってしまうので、select.phpの
　右上の「id_Reset」をクリックすると、idが1から振り直される。
-写真を登録できるようにした
